# 🧠 Retail Intelligence Dashboard

A full-stack Machine Learning powered analytics system built using **Flask, PostgreSQL, Pandas, Scikit-Learn, and Chart.js**.

This project allows users to upload retail sales data and automatically generate:

- 📊 Revenue analytics
- 📈 Trend visualization
- 🤖 ML-based revenue prediction
- 🏆 Category-wise forecast
- 📄 PDF export report
- 🌙 Light/Dark mode UI
- 🔁 Manual model retraining

---

## 🚀 Features

### 🔐 Authentication
- User Registration & Login
- Secure session management

### 📂 Data Upload
- Upload `.csv` or `.xlsx` files
- Automatic database insertion
- Dynamic aggregation

### 📊 Analytics Dashboard
- Total Revenue
- Total Orders
- Monthly Revenue Trend (Animated Graph)
- Forecasted Revenue

### 🤖 Machine Learning
- Linear Regression
- Random Forest Regressor
- Automatic Best Model Selection (based on R²)
- Next Month Revenue Prediction
- Category-wise Revenue Forecast
- Model Accuracy Comparison
- Manual Model Retraining
- Model Training Timestamp

### 📄 Export Report
- Download full analytics report as PDF
